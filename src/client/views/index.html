Natural Language Processing

Natural Language Processing which is also known as NLP, is used to understand and process the natural language a human speaks. It also gives us results accordingly for that entered text. Amazingly, we can also check if the entered text is positive, negative or nuetral. Which we are testing in this project along with couple of other things.

We are using Aylien API which analysis text entered by the user and shows us the power of Natural Language Processing on a bigger picture. It understands what or who a document mentions, what topics it deals with, and what the sentiment of a piece of text is.
